## RU: Кизер — ИИ-экскурсовод (этап 1)

Минимальный прототип мобильного интерфейса «Кизер — ИИ-экскурсовод» на Next.js (App Router) + TypeScript + Tailwind.

- **Реализовано**: структура проекта под режимы `/questions`, `/photo`, `/geo`, `/settings`, `/admin`; главная страница с тремя режимами и нижней панелью; простые страницы-заглушки; README-файлы `aaa_*.txt` по папкам.
- **UI**: mobile-first, фиксированная ширина контейнера (до 390px), центрирование, стилистика в духе iOS (воздух, скругления, тени), тексты на русском.

**Как запустить**

```bash
npm install
npm run dev
```

Откройте `http://localhost:3000` в браузере. На ПК интерфейс выглядит как экран телефона по центру окна.

## EN: Kizer — AI tour guide (phase 1)

Minimal mobile-style prototype of “Kizer — AI tour guide” built with Next.js (App Router) + TypeScript + Tailwind.

- **Implemented**: project structure for `/questions`, `/photo`, `/geo`, `/settings`, `/admin`; main page with three modes and a bottom bar; simple stub pages; `aaa_*.txt` README files per folder.
- **UI**: mobile-first, fixed container width (up to 390px), centered layout, iOS-like minimalist styling, Russian interface texts.

**How to run**

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser. On desktop it renders as a centered phone-like screen.
